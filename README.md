
## Usage 


#### 1. clone and install requirements
```bash
$ pip install -r requirements.txt
```

#### 3. Preprocessing

```bash
$ python build_vocab.py   
$ python resize.py
```

#### 4. Train the model

```bash
$ python train.py    
```

#### 5. Test the model 

```bash
$ python sample.py --image='png/example.png'
```
